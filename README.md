# 🛡️ Phishing Awareness Analysis

## 📌 Project Overview
This project simulates the role of a Cybersecurity Analyst tasked with identifying, analyzing, and responding to social engineering threats. The goal is to bridge the gap between human error and system security by building a robust "Human Firewall."

## 📂 Project Deliverables
| Document | Description |
| :--- | :--- |
| [Phishing Triage Checklist](Checklist.doc) | A non-expert cheat-sheet of 15+ red flags to spot scams instantly. |
| [Decision Tree](DecisionTree.doc) | A visual flowchart guiding users to classify emails as Safe, Suspicious, or Malicious. |
| [Email Forensic Analysis](EmailAnalysis.doc) | In-depth breakdown of a sophisticated Business Email Compromise (BEC) / Whaling attempt. |

## 🧠 Key Skills Demonstrated
- Threat Identification & Triage
- Social Engineering Psychology (Urgency, Authority, Fear)
- Domain Spoofing & Header Analysis (SPF, DKIM, DMARC)
- Incident Response (Pause, Verify, Report)

## 🔍 Case Study Analyzed
Analyzed a malicious email impersonating a CEO requesting an urgent, confidential wire transfer. Classified as **MALICIOUS** due to domain mismatch, bypass requests, and extreme urgency.
